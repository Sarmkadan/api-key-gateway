# CacheKeyGeneratorValidation

`CacheKeyGeneratorValidation` provides validation utilities for cache keys generated by the API Key Gateway. It ensures that generated keys conform to expected patterns and constraints before they are used in caching operations, preventing runtime errors and security issues.

## API

### Methods

#### `Validate(string key)`
Validates a single cache key and returns a list of validation error messages.
- **Parameters**
  - `key` (string): The cache key to validate.
- **Returns**
  - `IReadOnlyList<string>`: A list of error messages; empty if the key is valid.
- **Throws**
  - `ArgumentNullException`: If `key` is `null`.

#### `Validate(IEnumerable<string> keys)`
Validates a collection of cache keys and returns a list of validation error messages.
- **Parameters**
  - `keys` (IEnumerable<string>): The collection of cache keys to validate.
- **Returns**
  - `IReadOnlyList<string>`: A list of error messages; empty if all keys are valid.
- **Throws**
  - `ArgumentNullException`: If `keys` is `null`.

#### `Validate(params string[] keys)`
Validates a variable number of cache keys and returns a list of validation error messages.
- **Parameters**
  - `keys` (params string[]): The cache keys to validate.
- **Returns**
  - `IReadOnlyList<string>`: A list of error messages; empty if all keys are valid.
- **Throws**
  - `ArgumentNullException`: If `keys` is `null`.

#### `IsValid(string key)`
Determines whether a single cache key is valid.
- **Parameters**
  - `key` (string): The cache key to check.
- **Returns**
  - `bool`: `true` if the key is valid; otherwise, `false`.
- **Throws**
  - `ArgumentNullException`: If `key` is `null`.

#### `IsValid(IEnumerable<string> keys)`
Determines whether all cache keys in a collection are valid.
- **Parameters**
  - `keys` (IEnumerable<string>): The collection of cache keys to check.
- **Returns**
  - `bool`: `true` if all keys are valid; otherwise, `false`.
- **Throws**
  - `ArgumentNullException`: If `keys` is `null`.

#### `EnsureValid(string key)`
Validates a single cache key and throws an exception if invalid.
- **Parameters**
  - `key` (string): The cache key to validate.
- **Throws**
  - `ArgumentNullException`: If `key` is `null`.
  - `ValidationException`: If the key is invalid.

#### `EnsureValid(IEnumerable<string> keys)`
Validates a collection of cache keys and throws an exception if any are invalid.
- **Parameters**
  - `keys` (IEnumerable<string>): The collection of cache keys to validate.
- **Throws**
  - `ArgumentNullException`: If `keys` is `null`.
  - `ValidationException`: If any key in the collection is invalid.
